# Vue-FlatUI
A Flat UI component library based on vue2.0 [Flat-UI Website](http://designmodo.github.io/Flat-UI/docs/components.html)
![preview](./preview.png)

## Usage
- git clone git@github.com:gongph/Vue-FlatUI.git 
- npm i yarn -g 
- yarn install
- yarn run dev
